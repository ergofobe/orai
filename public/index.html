orai

OpenRouter AI from the command line

Install with one command
curl -fsSL https://ergofobe.github.io/orai/install.sh | sh
Or with Homebrew (macOS)
brew install ergofobe/orai
🤖

Any Model

Use any model on OpenRouter. GPT-4o, Claude, Gemini, Llama, and 200+ more.

🔧

Native Tools

Read files, write files, run shell commands, and fetch URLs. AI-powered coding right in your terminal.

📎

Attachments

Attach images, PDFs, and code files. PDFs auto-convert to images for vision models.

Quick Start

orai prompt "Explain Rust's borrow checker"
One-shot prompt with default model (openrouter/free)
orai -m anthropic/claude-3.5-sonnet prompt "Review this code"
Use a specific model
orai -a screenshot.png prompt "What does this show?"
Attach an image
orai chat
Interactive REPL conversation
orai tui
Full terminal UI with markdown rendering

Tools

read(path)
Read file contents from the local filesystem
write(path, content)
Write content to a file ⚠ requires confirmation
shell(command)
Execute shell commands ⚠ requires confirmation
web_fetch(url)
Fetch content from a URL
openrouter:web_search
Search the web for current information (server-side)
openrouter:datetime
Get the current date and time (server-side)
